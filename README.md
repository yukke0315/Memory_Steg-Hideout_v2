# Memory Steg-Hideout v2

### 画像のピクセル内に思い出を暗号化して隠す、ステガノグラフィー技術を用いた完全クライアントサイドの秘密のギャラリーアプリ。

## 概要

「Memory Steg-Hideout」は、大切な写真とその日の日記・思い出を、第三者にはただの画像にしか見えない形で保存・共有できるアプリケーションです。画像のRGBAデータのうち、RGBの最下位ビット（LSB）を書き換えるLSBステガノグラフィー技術を用いて、テキスト（JSON）データをバイナリ変換しピクセル内に埋め込みます。すべての暗号化・復号化およびZIP生成処理はブラウザのメモリ上（クライアントサイド）のみで完結するため、サーバーへのデータ送信は一切発生せず、極めてセキュアに秘密の思い出を管理できます。

## 主な機能

### Create Hideout (暗号化＆生成画面)
- **データの埋め込み**: ユーザーがアップロードした画像と、入力した「タイトル・日付・日記」をひとつのJSONにまとめ、LSBステガノグラフィーで画像のピクセルに埋め込みます。
- **ブラウザの補正回避処理**: Canvas APIの仕様によるAlpha（透明度）の自動補正で暗号が破壊されるのを防ぐため、RGBチャンネルのみを計算して正確な暗号化を実現しています。
- **ZIP一括エクスポート**: JSZip を活用し、暗号化された画像（可逆圧縮のPNG形式）をブラウザ上で即座にZIPファイル（Hideout）として生成・ダウンロードします。

### Step Into Hideout (復号＆ギャラリー画面)
- **ドラッグ＆ドロップ解凍**: エクスポートしたZIPファイルをドロップするだけで、ブラウザ上で即座に解凍・抽出が行われます。
- **リアルタイム暗号解読**: 画像のピクセルからバイナリデータを抽出し、元のJSONデータを一文字の欠損もなく復元します。
